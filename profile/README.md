Citrix DaaS delivers secure virtual apps and desktops from the cloud, helping IT manage access, scale hybrid work, and simplify user experiences.

Citrix DaaS is a desktop as a service platform for publishing Windows apps, complete desktops, and remote work resources through the Citrix cloud control plane. Teams use Citrix DaaS login workflows to connect users with assigned resources while administrators manage images, policies, access, and monitoring from a centralized service.

The platform is designed for organizations that need flexible access without rebuilding every endpoint strategy.
